# Ruralit – Libreta del Campo

Tu libreta digital para el campo. Registrá gastos, ingresos y llevá el balance de tu establecimiento de forma simple y profesional.

## Características
- Gestión de múltiples establecimientos.
- Registro rápido de movimientos con lenguaje natural.
- Balance detallado mensual y anual.
- Gráficos interactivos de evolución financiera.
- Offline-first con IndexedDB.

## Desarrollo
```bash
npm install
npm run dev
```
